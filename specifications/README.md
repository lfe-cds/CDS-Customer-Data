# CDS Customer Data Working Group (WG3) Specifications

This is the location that contains the specifications for the CDS Customer Data Working Group (WG3) (see the `.md` files in this folder).

## Notes
* Prior July 2024, the Customer Data working group encompassed all Registration and Customer Data specifications and was called CDSC Customer Data WG1.
* In July 2024, WG1 was renamed to Connectivity WG1 and continued to maintain the first two specifications (Server Metadata and Client Registration), and Customer Data was reorganized into to Customer Data WG3 to maintain the third specification (Customer Data).
    * [Server Metadata](https://cds-registration.lfenergy.org/specs/cds-wg1-01/) specification (`CDSC-WG1-01`) was moved to the newly formed [Connectivity WG1](https://cds-registration.lfenergy.org/).
    * [Client Registration](https://cds-registration.lfenergy.org/specs/cds-wg1-02/) specification (`CDSC-WG1-02`) was moved to the newly formed [Connectivity WG1](https://cds-registration.lfenergy.org/).
    * [Customer Data](https://cds-customerdata.lfenergy.org/specs/cds-wg3-01/) specification was renamed to match the updated Customer Data WG3 name (`CDSC-WG1-03` --&gt; `CDSC-WG3-01`).
* In July 2025, the project was renamed from CDSC to CDS (Connected Data Specifications), the Connectivity working group (WG1) was renamed to the [Registration working group](https://cds-registration.lfenergy.org/) (WG1).
    * [Server Metadata](https://cds-registration.lfenergy.org/specs/cds-wg1-01/) specification was renamed to match the updated CDS name (`CDSC-WG1-01` --&gt; `CDS-WG1-01`).
    * [Client Registration](https://cds-registration.lfenergy.org/specs/cds-wg1-02/) specification was renamed to match the updated CDS name (`CDSC-WG1-02` --&gt; `CDS-WG1-02`).
    * [Customer Data](https://cds-customerdata.lfenergy.org/specs/cds-wg3-01/) specification was renamed to match the updated CDS name (`CDSC-WG3-01` --&gt; `CDS-WG3-01`).
